# Harsh Kumar Shaw | GenAI Engineer & Researcher

A premium, high-performance portfolio showcasing production-grade GenAI systems, multi-agent pipelines, and AI research in drug discovery.

🚀 **Live Site:** [https://github.com/harshkumarshaw/genai-portfolio](https://github.com/harshkumarshaw/genai-portfolio) (Deploy to Vercel for live link)

## 🛠️ Tech Stack
- **Framework:** Next.js 15 (App Router, Turbopack)
- **Styling:** Vanilla CSS (Modular) with Premium Design System
- **Animation:** Framer Motion
- **Icons:** Lucide React
- **Data:** JSON-driven content architecture
- **Content:** MDX for Case Studies

## 🌟 Key Features
- **Premium Design System:** Custom dark/light mode with smooth transitions and glassmorphism.
- **Product-Focused Case Studies:** Deep dives into engineering decisions (NL2BQ, HCC-AI, VaultCall).
- **Research Integration:** Dedicated section for peer-reviewed publications (Springer, ICEMSSD).
- **Build Timeline:** Interactive reverse-chronological development history.
- **Mobile Optimized:** Fully responsive experience across all devices.

## 📁 Project Structure
- `/src/data`: Content source of truth (JSON)
- `/src/components`: Modular React components
- `/content/projects`: MDX case studies
- `/public`: Assets and resume variants

## 🚀 Getting Started
```bash
npm install
npm run dev
```
