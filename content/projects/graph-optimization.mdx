## Overview

An algorithmic systems project focused on computing optimal radial network configurations by implementing the Reverse Delete algorithm to find the Minimum Spanning Tree (MST) of a graph.

## Problem

In network design (such as telecommunications, power grids, or computer networks), minimizing the total cost of connections while ensuring all nodes remain connected is a classic optimization problem. Finding the most efficient algorithmic approach is critical for scaling these networks.

## Why This Approach ⭐

While Kruskal's or Prim's algorithms are standard for finding MSTs, implementing the **Reverse Delete algorithm** provided a unique perspective on graph optimization. It works backwards—starting with the full graph and iteratively deleting the most expensive edges, provided the deletion doesn't disconnect the graph.

## Solution + Architecture

- **Domain**: Algorithms and Data Structures.
- **Tech Stack**: Implemented using core computer science languages (C++/Python), focusing purely on algorithmic efficiency.
- **Techniques**: Graph theory, cycle detection, and edge sorting.

## Results / Impact

- Successfully computed the minimum spanning solution for complex test graphs.
- Demonstrated strong foundational capabilities in algorithmic problem-solving, theoretical computer science, and performance optimization.
